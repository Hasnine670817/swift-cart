**1) What is the difference between null and undefined?**
**Ans:** When a variable is declared but not assigned a value, it is undefined
and `null` is an intentional empty value assigned by the developer.

**2) What is the use of the  `map()`  function in JavaScript? How is it different from  `forEach()`?**
**Ans:** map() operates on each element of the array and returns a new array, while forEach() does not return an array.

**3) What is the difference between  `==`  and  `===`?**
**Ans:** `==` only checks the value and  `===` checks both the value and type.

**6) Explain the concept of Scope in JavaScript (Global, Function, Block).**
**Ans:** Variables that are declared outside a function or block are in global scope, variables that are declared inside a function and cannot be accessed from outside the function are in function scope, and variables that are declared inside `{}` (such as if, for, while) are in block scope.
